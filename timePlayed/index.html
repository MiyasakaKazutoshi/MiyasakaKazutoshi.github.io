# 概要 ・テレビゲームのプレイ時間のように、クエストをプレイした時間を記録するアプリです。 # クエストについて ・クエスト名は自由に入力できます。(e.g. 勉強、 筋トレ、音楽、柔軟 ) ・クエストはいくつでも作成できます。 ・一度に一つのクエストのみ実行中にできます。 # ドラゴンについて ・クエストをプレイした合計時間がドラゴンの経験値となります。 ・ドラゴンは約100時間で進化し、5回の進化で完全体になります。 ・4体のドラゴンを完全体にした時、最強のドラゴンが誕生するでしょう。(約2000時間) # 機能 ・機種変更の際にデータを引き継ぐためのオンラインバックアップが可能です。 ・クエストごとのプレイ時間をグラフで見ることができます。 # その他 ・アプリが好評な場合、各属性の進化形態が増えます！！
