# 概要\n ・テレビゲームのプレイ時間のように、クエストをプレイした時間を記録するアプリです。\n # クエストについて\n ・クエスト名は自由に入力できます。(e.g. 勉強、 筋トレ、音楽、柔軟 )\n ・クエストはいくつでも作成できます。\n ・一度に一つのクエストのみ実行中にできます。\n # ドラゴンについて\n ・クエストをプレイした合計時間がドラゴンの経験値となります。\n ・ドラゴンは約100時間で進化し、5回の進化で完全体になります。\n ・4体のドラゴンを完全体にした時、最強のドラゴンが誕生するでしょう。(約2000時間)\n # 機能\n ・機種変更の際にデータを引き継ぐためのオンラインバックアップが可能です。\n ・クエストごとのプレイ時間をグラフで見ることができます。\n # その他\n ・アプリが好評な場合、各属性の進化形態が増えます！！\n
